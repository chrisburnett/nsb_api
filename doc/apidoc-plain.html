Assignments / update

PATCH /api/v1/jobs/:job_id/assignments/:id
Update a particular assignment

PUT /api/v1/jobs/:job_id/assignments/:id
Update a particular assignment

PATCH /api/v1/assignments/:id
Update a particular assignment

PUT /api/v1/assignments/:id
Update a particular assignment

Params

  • assignment : optional [

    Must be a Hash

    ]

    Assignment properties

    • contractor_id : optional [

      Must be a String

      ]

      ID of Contractor (User) associated with this assignment

    • job_id : optional [

      Must be a String

      ]

      ID of Job associated with this assignment

    • am_pm_visit : optional [

      Must be one of: am, pm.

      ]

      Whether assignment is AM or PM visit.

    • resolution : optional [

      Must be a String

      ]

      Resolution

    • notes : optional [

      Must be a String

      ]

      Supplementary notes

    • scheduled_date : optional [

      Must be a String

      ]

      Date assignment scheduled to be done

    • actual_date : optional [

      Must be a String

      ]

      Date assignment was actually fulfilled. NOTE: not set automatically on state change to fulfilled, but it should be

    • status : optional [

      Must be one of: accepted, rejected, fulfilled.

      ]

      Status of the assignment

    • signature : optional [

      Must be a String

      ]

      Encoded image data - signature image upload
